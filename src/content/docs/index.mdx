## Qué es esto

Un curso completo, navegable y de calidad universitaria que te lleva desde no
saber programar hasta ser **AI / Automation Engineer semi-senior**. Pensado para
un **cero real**: cada concepto se enseña desde sus fundamentos, con la
experiencia previa marcada como _opcional_, nunca como prerrequisito.
